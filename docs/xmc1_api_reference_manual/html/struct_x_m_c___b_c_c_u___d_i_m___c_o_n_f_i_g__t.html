Detailed Description

Configures dimming engine settings of the BCCU module.

Field Documentation

◆ cur_sel

uint32_t cur_sel

Selects a type of exponential curve. Use type XMC_BCCU_DIM_CURVE_t. If dither
enabled, the configuration is being ignored

◆ dim_div

uint32_t dim_div

Configures a dimming clock divider, used to adjust the fade rate. If 0, the dimming level
as same as target dimming level on shadow transfer

◆ dither_en

uint32_t dither_en

Selects a dither enable. Dithering added for every dimming step if dimming level < 128.
