Detailed Description

Defines a data structure for initializing the data of the clock functional block. Clock functional block configures clock dividers, peripheral and RTC clock source by configuring corresponding bits in CLKCR clock control register. Use type XMC_SCU_CLOCK_CONFIG_t for accessing these structure parameters.
