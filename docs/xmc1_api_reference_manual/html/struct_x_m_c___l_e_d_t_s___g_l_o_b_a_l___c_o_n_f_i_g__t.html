XMC_LEDTS_GLOBAL_CONFIG_t Struct Reference
LED and Touch-Sense control(LEDTS)

#include <xmc_ledts.h>

Detailed Description

Data structure for initialization of global features common to LED and touch-sense function. Use type XMC_LEDTS_GLOBAL_CONFIG_t for this structure.

Field Documentation

◆ autoscan_synchronization

uint32_t autoscan_synchronization

Set this bit to synchronize start of autoscan time period with master kernel(ENSYNC). Refer XMC_LEDTS_TP_SYNC_t enum for possible values.

◆ clock_generation

uint32_t clock_generation

When this bit is set LEDTS counter takes its clock from another master kernel. Kernel generates its own clock when this bit is not set (CMTR). Refer XMC_LEDTS_CLOCK_TYPE_t enum for possible values.

◆ suspend_response

uint32_t suspend_response

Suspend request configuration(SUSCFG). Refer XMC_LEDTS_SUSPEND_t enum for possible values.
