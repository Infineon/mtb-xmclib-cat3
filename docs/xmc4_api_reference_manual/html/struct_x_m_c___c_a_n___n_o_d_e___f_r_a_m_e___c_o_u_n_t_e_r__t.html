Detailed Description

Defines frame counter configuration. Use type XMC_CAN_NODE_FRAME_COUNTER_t for this structure. It provides configuration of frame counter that counts transmitted/received CAN frames or obtains information about the time when a frame has been started to transmit or be received by the CAN node.
