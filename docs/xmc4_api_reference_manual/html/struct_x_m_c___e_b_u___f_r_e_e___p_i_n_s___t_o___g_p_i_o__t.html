XMC_EBU_FREE_PINS_TO_GPIO_t Struct Reference
External Bus Unit (EBU)

#include <xmc_ebu.h>

Detailed Description

GPIO mode configuration for the allocated EBU ports
Configuring this structure frees the allocated EBU ports for GPIO functionality. The XMC_EBU_Init() is used to populate the structure with the GPIO mode for the allocated EBU ports.
