Delta Sigma Demodulator (DSD)

#include <xmc_dsd.h>

Detailed Description

DSD timestamp saves result, filter counter register and integrator count register. Note: Trigger source for timestamp and integrator trigger are shared. DSD Filter is mandatory to use this block.
